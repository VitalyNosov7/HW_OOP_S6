## Реструктуризировать проект homework с учетом теоретических основ SOLID.
---
## Решение:
1. Добавлен Класс Controller (пакет Controller) в котором инкапсулированы методы создания и сохранения отчета пользователя;
2. Добавлен Класс View (пакет View) в котором реализован вывод информации;
3. Добавлен пакет Model в который собраны Классы Persister, Report и User;
4. Из Класса User метод report() был перенесен в отдельный Класс - Report;
5. Из Класса User вызов метода save() (Класса Persister) был перенесен в Класс Controller (пакет Controller);
6. Из Класса User вызов метода report() (Класса Report) был перенесен в Класс Controller (пакет Controller);
7. В клиенсткой части осталось создание экземпляра класса User, который передается в созданный экземпляр класса Controller, в котором производится вызов методов, которые создают и сохраняют отчет.
